Weather Forecast

HTML5, CSS3, jQuery, and RESTful API's were used for this weather app. The basic function for this is that it detects the user's location and returns the current weather for that location, along with an icon of the condition and a button that converts from °F to °C.

Pomodoro Timer

A pomodoro timer is used to increase productivity. It has two timers, one is a work session where the user stays concentrated and working throughout that session, and a break session where the user takes a break. Once the work session expires the break session starts, and the timer keeps alternating between one and the other. HTML5, CSS3, and jQuery were used to develop this application.
